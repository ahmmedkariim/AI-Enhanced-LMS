## 5. Authentication & Roles

### Roles
| Role | Permissions |
|---|---|
| Student | access courses, quizzes |
| Instructor | manage content, monitor |
| Admin | system control |
| AI Service | automated analysis |
| LLM Service | content generation |

### BPMN Authentication Flow
- Start Event → Login Request
- User Task: Enter Credentials
- Service Task: Validate Credentials
- **Exclusive Gateway:** Valid credentials?
  - Yes → Assign Role
  - No → Show error → Retry login
- **Role-Based Gateway:**
  - Student → learning process
  - Instructor → teaching dashboard
  - Admin → system management

---

## 6. Required BPMN Gateways & Tasks

### Gateways
1. **Exclusive Gateway** — pass/fail quiz, login success/failure
2. **Parallel Gateway** — AI analysis + LLM generation simultaneously
3. **Event-Based Gateway** — waiting for student response OR timeout
4. **Inclusive Gateway** — assigning multiple learning resources simultaneously

### Required User Tasks (minimum)
- Login
- Submit learning goals
- Participate in classroom activity
- Answer live quiz
- Take adaptive quiz
- Review feedback
- Collaborate in team challenge

## 7. DMN Decision Tables (4 Required)

### DMN 1: Learning Path Decision
- **Inputs:** Quiz Score, Learning Style, Engagement Level
- **Output:** Next Learning Path

| Score | Style | Engagement | Output |
|---|---|---|---|
| High | Visual | High | Advanced Visual Content |
| Low | Reflective | Low | Remedial Content |
| Medium | Active | High | Practice Exercises |

### DMN 2: Gamification Reward Decision
- **Inputs:** Activity Type, Performance, Collaboration Level
- **Output:** Reward Type

| Activity | Performance | Output |
|---|---|---|
| Quiz | High | Badge |
| Team Task | High | Collaboration Reward |
| Exploration | Medium | Hidden Reward |

*(2 more DMN tables needed — can be designed as part of the project)*

## 8. Hybrid Lecture Process

### Before Lecture
- Service Task: AI analyzes class profile
- Service Task: LMS prepares adaptive lecture
- Parallel Gateway:
  - Generate content
  - Generate in-class quiz

### During Lecture
- User Task: Student participates in poll
- User Task: Student joins team challenge
- User Task: Student answers live quiz
- **Event-Based Gateway:** Wait for student response OR timeout
- **Inclusive Gateway:** Trigger multiple → reward points + feedback + leaderboard update

### After Lecture
- Service Task: AI summarizes lecture
- Service Task: LLM generates adaptive quiz
- User Task: Student completes quiz
- **Gateway:** Pass?
  - Yes → Reward
  - No → Remedial content
